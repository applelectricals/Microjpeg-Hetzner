How It Works

User saves a product in JoomShopping admin
Plugin triggers onAfterSaveProduct event
Plugin sends image to MicroJPEG API for compression
Compressed image is downloaded and replaces original
Success message shows: "MicroJPEG: 3 image(s) optimized, 45.2 KB saved!"

Installation

Upload ZIP in Joomla: System → Install → Extensions
Enable plugin: System → Plugins → search "MicroJPEG"
Configure: Click plugin → enter API key
Done! Images now auto-optimize on save
